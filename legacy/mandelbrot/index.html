The Mandelbrot Set

The mandelbrot set is the set of complex numbers for which the equation for Z

Z0 = C
Zn+1 = Zn2 + C

is bounded (does not trend to infinity) where C is a complex point

These points are plotted black on the left if they are in the set, and those that are not in the set are colored based on the amount of iterations it takes for Z to "break out"


The plot on the left is interactive

  • You can adjust the scale with the numeric input
  • You can move the plot by left clicking and dragging
  • You can see the sequence of Z's that a point makes by right clicking
